Helpers

throwErrnoIfMinus1NoRetry :: (Eq a, Num a) => String -> IO a -> IO a Source #

Throw an IOError corresponding to the current value of getErrno if the result value of the IO action is -1 and getErrno is not eINTR. If the result value is -1 and getErrno returns eINTR 0 is returned. Otherwise the result value is returned.

exchangePtr :: Ptr (Ptr a) -> Ptr a -> IO (Ptr a) Source #

exchangePtr pptr x swaps the pointer pointed to by pptr with the value x, returning the old value.
